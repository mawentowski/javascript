Payments API

The Payments API provided by YourCo allows developers to create and manage charges. Full documentation is available at yourdocs.example.com/payments.

Create a Charge (POST /v1/charges)

To create a charge, send a POST request to:

https://api.example.com/v1/charges

The request must include amount, currency, and source fields in application/json format.

  • OpenAPI 3.1 Spec
  • Swagger UI
  • Postman Collection

How to: Create a Charge via the Payments API

  1. Review API terms: You must accept the API Terms of Use before creating charges.
  2. Examine the operation schema: Check required fields and response semantics in the OpenAPI spec.
  3. Create the charge: Send a POST request to the endpoint using a tool like curl or Postman.

Code Example (curl)

curl -X POST https://api.example.com/v1/charges \
  -H 'Authorization: Bearer YOUR_KEY' \
  -H 'Content-Type: application/json' \
  -d '{"amount":100,"currency":"USD","source":"tok_visa"}'
